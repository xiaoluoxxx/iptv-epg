
## 简介

PTV直播源码，适用于电视直播。

## 如何使用

### 文件说明

- `tv-ipv4-old` => 参考 [BurningC4](https://github.com/BurningC4/Chinese-IPTV) 的仓库，两年前的原始数据，部分可用，存在延迟和卡顿

- `epg.xml` => 由 `get-epg.py` 抓取数据并生成，通过 Actions 每天定时自动更新

- `requirements.txt` => Python 程序 `get-epg.py` 的依赖包
